# 🍕 Pizza Shop API - Order System

This is a simple and secure API to manage orders for a pizza shop. It handles user signup, login, and order tracking.

---

## 🛠️ Technologies Used

| Technology | What does it do |
| :--- | :--- |
| **FastAPI** | Main framework, to create the API |
| **SQLAlchemy** | ORM. Connects the Python code to the database. |
| **JWT (python-jose)** | Security. Creates tokens to keep users logged in. |
| **Passlib (Bcrypt)** | Hashes and secures user passwords. |
| **Pydantic** | Validates data formats. |
| **Uvicorn** | The server used to run the application. |

---

## 🚀 Main Features

* **Authentication:** User registration, secure login, and token refresh.
* **Order Management:** Create and view customer orders.
* **User Roles:** Separates regular customers from administrators (admin).
